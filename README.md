# Bhavik ❤️ Pooja | Wedding Invitation 💍

A premium, cinematic, and fully responsive digital wedding invitation website built with **React**, **Vite**, and **Framer Motion**. This project features a traditional Gujarati wedding theme with modern aesthetics like glassmorphism, paper textures, and intricate gold filigree.

## ✨ Project Highlights

### 🏛️ Ancestral Blessings
A deeply respectful section honoring family roots with an "Arched Shrine" design, gold filigree frames, and gentle marigold petal animations.

### 📜 Sacred Ceremonies
A beautiful, staggered timeline showcasing the journey of celebrations—from Haldi to the Wedding Lunch—adorned with ornate decorations and micro-animations.

### 🖼️ Memories Gallery
A curated collection of special moments presented in a premium masonry-style grid with smooth transitions.

### 📍 Venue & Map
Precise location details with integrated Google Maps to guide guests to the celebrations at Laxmi Villa, Surat.

## 🛠️ Tech Stack

- **Frontend:** React.js, Tailwind CSS
- **Animations:** Framer Motion (Scroll-triggered animations, floating petals, spring physics)
- **Typography:** Playfair Display, Cinzel, Great Vibes
- **Build Tool:** Vite
- **SEO:** React Helmet Async

## 🎨 Design System

- **Primary Colors:** Wedding Gold (#D4AF37), Traditional Maroon (#4A0404), Cream Paper (#FFFDFC).
- **Aesthetics:** Glassmorphism, Paper Texture Overlays, Arched Shrines, Gold filigree.
- **Responsiveness:** Mobile-first design with complex staggered layouts for desktop.

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone [repository-url]
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Run the development server:**
   ```bash
   npm run dev
   ```
4. **Build for production:**
   ```bash
   npm run build
   ```
